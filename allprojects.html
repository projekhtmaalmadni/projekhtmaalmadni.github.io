Qara’a Release

Make Learn Qur'an Fun With Qara’a

Qara'a is an interactive and modern digital Al-Qur'an learning platform.

Project Overview

Qara'a is an innovative educational application that combines traditional Islamic learning with modern digital technology. The platform offers interactive Quran learning, prayer time tracking, and comprehensive Islamic educational content.

Interactive Quran learning with audio recitation
Prayer time notifications and Qibla direction
Islamic calendar and important dates

Technical Implementation

Frontend

  • • React Native for cross-platform
  • • Tailwind CSS for styling
  • • Redux for state management
  • • React Navigation

Backend

  • • Node.js with Express
  • • PostgreSQL database
  • • RESTful API architecture
  • • JWT authentication

Features

  • • Offline mode capability
  • • Push notifications
  • • Audio streaming
  • • Location-based services
